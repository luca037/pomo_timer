# pomo_timer
A pomodoro timer written in C++.
### Setup 
Clone repo:
```bash
git clone https://github.com/luca037/pomo_timer.git
```
Build project:
```bash
mkdir build && cd build
cmake ../
cmake --build .
```
Track your daily pomos:
```bash
touch calendar.txt
```
Add exec permission:
```bash
chmod +x pomo.sh
```
Run pomo timer:
```bash
./pomo.sh
```
